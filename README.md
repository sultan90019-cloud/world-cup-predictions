# World Cup Predictions

نظام مسابقة توقعات كأس العالم مع تسجيل دخول ومتسابقين، جدول مباريات، لوحة مدير لتسجيل النتائج، وترتيب تلقائي.

## المتطلبات
- Node.js مثبت على الجهاز (يفضل النسخة LTS مثل 18 أو 20)
- npm متاح مع Node.js

## كيفية التشغيل
1. افتح موجه الأوامر أو PowerShell في المجلد:
   `c:\Users\ME\OneDrive - Business Sultan\سطح المكتب\world cup`
2. نفذ الأمر:
   `npm install`
3. ثم شغّل التطبيق:
   `npm start`
4. افتح المتصفح وادخل:
   `http://localhost:3000`

## بيانات تسجيل دخول المدير
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

## الوظائف المتوفرة
- تسجيل متسابق جديد باسم وكلمة مرور.
- إدخال توقعات المباريات لكل متسابق.
- إغلاق التوقع تلقائياً قبل بداية المباراة بـ 15 دقيقة.
- صفحة مسؤول لتسجيل النتائج الفعلية.
- حساب النقاط تلقائياً وفقاً للقواعد:
  - توقع النتيجة بالضبط = 20 نقطة.
  - توقع الفائز الصحيح فقط = 10 نقاط.
  - توقع الفائز الصحيح وفارق الأهداف الصحيح = 15 نقطة.
- عرض ترتيب المتسابقين وترتيبه بشكل تلقائي.
